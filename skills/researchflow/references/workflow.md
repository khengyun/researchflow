# ResearchFlow Workflow

ResearchFlow exists to stop agents from losing research context. The expected
cycle is:

1. Inspect existing experiment graph.
2. Search for similar old work.
3. Plan the new experiment with parent/citation links.
4. Run or inspect artifacts.
5. Compare against prior experiments.
6. Record improvements, regressions, decisions, and open risks.
7. Close the session and validate the graph.

The graph should answer:

- What did this experiment derive from?
- What did it cite as supporting evidence?
- What did it improve or regress?
- Which prompt or session proposed it?
- Which artifacts prove the result?
- Which old failed experiment should not be repeated?
